Settings.txt
============

List of settings used by KLineEd and some examples

Format
------

   --a 1 2 3 means parameter 'a' with arguments 1, 2 and 3.
   [--a ... | --b ...] means either parameter a or b. 
   --a [1 ... | 2 ...] means parameter a with either argument 1 or 2.
   (--c ...) means parameter c is optional. 
   --c 1 (2) means argument 2 is optional.
   --d 5 <min 0 max 10> means argument for parameter d is a number in range 0-10 with a default value of 5.
   --e 1 '2 x' 3 means there are three arguments for parameter e; 1, '2 x' and 3. The second argument is 2 x (with space between 2 and x).



Examples
--------



Setting List
------------

   {word} is the word currently selected by the cursor 
   COLOUR is any of the defined colour words; black, 
   > is any displayable character
   . is any displayable character
   drive:path\ is any valid full path
   [secret manager key] lookup value in secret manager using key value

--help														//display help info
--reset factory-defaults 'drive:path\xxx.json'			    //reset setitngs file to factory default 
--export 'drive:path\edit'.ksx 'drive:path\export.txt'		//export edit.ksx to export.txt
--import 'drive:path\import.txt' 'drive:path\edit.ksx'		//import import.txt to edit.ksx
--fix 'drive:path\edit.ksx' 'drive:path\new.ksx'			//fix errors in edit.ksx and write to new.ksx
--edit 'drive:path\edit.ksx'								//edit specified .ksx file
  --settings 'drive:path\xxx.json' (update)				    //name of settings file to read (note 1) or update (note2). These settings override the corresponding settings found in the global settings file (see note 3)
  
Edit parameter additional options - overrides those found in setting file(s)

  --svn username 'wills' password=[secret manager key] urlRoot 'https//me.svnrepository.com'
  --audiovol 0  <min 1 max 10>								//(0 is off)
  --audiofile (cr 'drive:path\bell.mpeg')  (key 'drive:path\click.mpeg') (startup 'drive:path\insertpaper.mpeg') (end 'drive:path\removepaper.mpeg') 
  --browser 'explorer.exe' - was ParamBrowser
  --urlhelp url https://github.com/wpqs/GenDotNetCmdApps/wiki/KLineEd-User-Manual-v1.0
  --urlsearch url https://www.google.com/ arg 'search/{word}'
  --urlthesaurus url https://www.thesaurus.com/ arg '/browse/{word}'
  --urlspell ulr https://www.spell.com arg 'search/{word}'
  --autocorrect [on | off]
  --rulers display [on | off] unit '.'
  --typingpause 60 <min 5 max 36000>						//in seconds
  --cursorsize 20 <min 1 max 100>
  --backgnd (text COLOUR) (msg-error COLOUR) (msg-warn COLOUR) (msg-note COLOUR) (cmds COLOUR) (status COLOUR) (rule COLOUR) //- was  ParamBackGndColour
  --foregnd (text COLOUR) (msg-error COLOUR) (msg-warn COLOUR) (msg-note COLOUR) (cmds COLOUR) (status COLOUR) (rule COLOUR)//- was  ParamForeGndColour
  --autosave [CR | ParaBreak | off]							//was ParamSpellCheckMode, spellcheck
  --scrollback 0  <min 1 max 10000>							//(0 is unlimited) - was scrollreview, ParamScrollReviewMode
  --editback  0  <min 1 max 10000>							//(0 is unlimited) - was editline (min current line) ParamEditLineMode
  --displaycols 80 <min 20 max 250>							//was DisplayLineWidth, now EditAreaLineWidth
  --displayrows 10 <min 1 max 25>                           //was displaylastlines, now EditAreaLinesCount
  --tabsize 3 <min 1 max 25>
  --parabreak char='>'

  Notes:
  1. the current command line parameters and args overwrite those found in the settings file
  2. the current command line parameters and args are used to update the settings file which is then read to define the settings for the editing session
  3. the priorty of settings is 1) any found in global settings file 2) any found in the specified file 3) any give on the command line
  4. settings file contains  ErrorReporting=[on | off] this allows disabling of MxReturnCode reporting
